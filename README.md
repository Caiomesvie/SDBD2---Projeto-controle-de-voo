# SDBD2---Projeto-controle-de-voo

Implementação de um Data Warehouse utilizando arquitetura em medalhões (Medallion Architecture). Trabalho da disciplina de Banco de Dados 2.

O projeto processa dados de voos de 2013 (CSV) desde a ingestão bruta até a modelagem dimensional (Star Schema).

## Tecnologias

* **Docker:** Containerização do ambiente.
* **PostgreSQL:** Banco de dados alvo (Data Warehouse).
* **PgAdmin:** Interface de gerenciamento do banco.
* **Apache Spark (PySpark):** Motor de processamento e ETL.
* **Jupyter Notebook:** Ambiente de desenvolvimento dos scripts.
* **Python:** Linguagem principal.
